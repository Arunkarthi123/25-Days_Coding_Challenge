# Jump searching :
    - Find sqrt of the length of array .
    - Segment the array in window size of the sqrt .
    - Check the last element of each window with the search value , If it is low than or equal to the last value then break otherwise continue .

# Interpolation Search :
    - As like binary Search , only formula changed . 
    - h , l , p , val 
    - p = l + {( h - l) / arr[h] - arr[l]} * ( x - arr[ l])
